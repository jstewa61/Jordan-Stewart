AROMA Coffee + Tea

Aroma Coffee + Tea is a coffee and tea shop in Kent and Columbus, Ohio potentially needing a new responsive website that will allow customers to find them, see the menu and learn about Aroma. I was involved in a highly collaborative project with one of my classmates to design, code and user test the new website.

Early on in the process, it was clear that the website should be made simple since most people will be viewing from their phone and the website will only need a few basic functions. The rough sketches show what some of the elements on the website ended up looking like. You can see the basic color palette and element styling we decided on that would work well with their existing brand.
